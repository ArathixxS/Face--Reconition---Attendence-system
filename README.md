# Face--Recognition--Attendance-System


# Overview
The Face Recognition Attendance System is an AI-powered attendance management solution that automatically detects and recognizes faces using computer vision techniques. The system captures facial images, identifies registered users, and records attendance efficiently, eliminating the need for manual attendance marking.

# Features

* User Registration with Face Capture
* Real-Time Face Detection and Recognition
* Automated Attendance Marking
* Attendance Record Storage
* User-Friendly Web Interface
* Secure Login and Registration System
* CSV-Based Attendance Management
